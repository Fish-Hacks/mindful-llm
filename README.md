# mindful-llm
LLM Service for MindfulHacks

## API Endpoints
### Summary
> **Link**: 127.0.0.1:5000/summary
``` json
[
    {
        "date": "2023-Jan-07",
        "emotion": "Happy",
        "description": "I am very happy today i passed my exam."
    },
...
]
```


### Chat
> **Link**: 127.0.0.1:5000/chat/{username}
``` json
{
  "message": "I'm feeling a bit down today :("
}
```
